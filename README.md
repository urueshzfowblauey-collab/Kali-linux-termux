## Qu'est-ce que c'est ?

**Kali-Termux** installe et gère un environnement Linux complet directement sur ton Android via Termux. Tu accèdes à un shell Ubuntu avec les outils de pentest les plus utilisés, le tout sans rooter ton téléphone.

---

## Installation rapide

```bash
pkg update && pkg upgrade -y && pkg install git -y
git clone https://github.com/TON_USER/Kali-linux-termux
cd Kali-linux-termux
chmod +x *.sh
bash kali.sh
```

Au premier lancement → **[1] Installer Linux** (~200 Mo).

---

## Menu principal

| Touche | Action |
|--------|--------|
| `1` | Lancer la session Linux |
| `2` | Mettre à jour Linux (apt) |
| `3` | Mettre à jour les outils |
| `4` | Installer un outil |
| `5` | Infos système |
| `6` | Sécurité rapide (scan, ping...) |
| `7` | Réinitialiser Linux |
| `8` | Désinstaller Linux |
| `9` | Updater le projet (git pull/push) |
| `b` | Backup (scripts + home + historique) |
| `0` | Quitter |

## Fonctionnalités

**Session Linux**
- Shell root complet via proot-distro
- Éditeur de texte (nano / vim)
- Logs & historique bash
- Infos système (OS, RAM, disk, IP)

**Sécurité rapide**
- Scan réseau local (nmap -sn)
- Scan de ports sur une IP
- Ping, traceroute
- Infos interface réseau

**Backup**
- Sauvegarde des scripts du projet
- Sauvegarde du home Linux (tar.gz)
- Sauvegarde de l'historique bash
- Listing et suppression des backups

**GitHub**
- git pull pour récupérer les mises à jour
- git push avec commit automatique horodaté
