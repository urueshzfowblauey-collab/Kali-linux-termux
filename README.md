## Installation

```bash
pkg update && pkg upgrade -y && pkg install git -y && git clone https://github.com/urueshzfowblauey-collab/Kali-linux-termux && cd Kali-linux-termux && chmod +x kali.sh config.sh setup_github.sh && bash kali.sh
```

---

## Utilisation

```bash
bash kali.sh
```

Au premier lancement, choisis **[1] Installer Linux**. L'installation télécharge l'environnement Ubuntu via proot-distro (~200 Mo). Une fois terminé, le menu principal s'affiche automatiquement.

### Menu principal

| Option | Action |
|--------|--------|
| `1` | Lancer la session Linux |
| `2` | Mettre à jour le système |
| `3` | Quitter |
| `tools` | Menu d'installation d'outils |
| `help` | Afficher l'aide |

---

## Outils disponibles

Accessible via la commande `tools` dans le menu :

| Outil | Description |
|-------|-------------|
| `nmap` | Scanner réseau |
| `sqlmap` | Injection SQL automatisée |
| `hydra` | Brute-force d'authentification |
| `nikto` | Scanner de vulnérabilités web |
| `john` | Cracker de mots de passe |
| `aircrack-ng` | Audit réseau Wi-Fi |
| `hashcat` | Cracker de hash GPU |
| `metasploit-framework` | Framework d'exploitation |
| `beef-xss` | Framework XSS |
| `gobuster` | Brute-force de répertoires |
| `dirb` | Scanner de contenu web |
| `netcat-openbsd` | Couteau suisse réseau |
