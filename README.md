## Qu'est-ce que c'est ?

**Kali-Termux** installe et gère un environnement Linux complet directement sur ton Android via Termux. Tu accèdes à un shell Ubuntu avec les outils de pentest les plus utilisés, des exercices OSINT interactifs, un gestionnaire VPN, un générateur de mots de passe, et bien plus — le tout sans rooter ton téléphone.

---

## Installation rapide

```bash
pkg update && pkg upgrade -y && pkg install git -y
git clone https://github.com/urueshzfowblauey-collab/Kali-linux-termux
cd Kali-linux-termux
chmod +x *.sh
bash kali.sh
```

Au premier lancement → **[1] Installer Linux** (~200 Mo).

## Fonctionnalités

**Session Linux**
- Shell root complet via proot-distro
- Éditeur de texte (nano / vim)
- Logs & historique bash
- Infos système (OS, RAM, disk, IP)
- Passage en utilisateur non-root

**Sécurité rapide**
- Scan réseau local (`nmap -sn`)
- Scan de ports sur une IP cible
- Ping et traceroute
- Infos interface réseau

**Backup**
- Sauvegarde des scripts du projet
- Sauvegarde du home Linux (`tar.gz`)
- Sauvegarde de l'historique bash Termux & Linux
- Listing, taille et suppression des backups

**Passwords** *(pass.sh)*
- Génération alphanumérique, avec symboles, numérique, etc.
- Choix de longueur (4–128) et nombre de mots de passe
- Génération de passphrases multi-mots
- Vérificateur de force avec score détaillé (7 critères)

**Clean** *(clean.sh)*
- Nettoyage cache apt + paquets orphelins (`autoremove`)
- Vidage des logs Linux (`syslog`, `auth.log`, etc.)
- Nettoyage Termux (`/tmp`, cache pkg)
- Suppression automatique des backups de plus de 7 jours
- Vue de l'utilisation disque par zone

**Doctor** *(doctor.sh)*
- Diagnostic Termux (bash, pkg, proot-distro, stockage, espace)
- Vérification de l'environnement Linux (OS, kernel, apt, espace)
- Présence et état de chaque outil (`nmap`, `sqlmap`, etc.)
- Vérification de tous les scripts (présence + exécutabilité)
- Test connectivité réseau et DNS
- Contrôle Git & remote GitHub
- Rapport final exportable en `.txt`

**Exercices OSINT** *(exercice.sh)*

12 challenges progressifs avec système de score (0–5 pts chacun) :

| # | Exercice | Difficulté |
|---|----------|------------|
| 01 | WHOIS — Infos domaine | ★☆☆☆☆ |
| 02 | DNS — Résolution d'adresses | ★☆☆☆☆ |
| 03 | TRACEROUTE — Cartographie réseau | ★★☆☆☆ |
| 04 | NMAP — Scan de base | ★★☆☆☆ |
| 05 | HTTP Headers — Fingerprinting serveur | ★★☆☆☆ |
| 06 | ROBOTS.TXT — Reconnaissance web | ★★☆☆☆ |
| 07 | GEO-IP — Localisation d'adresses | ★★★☆☆ |
| 08 | Métadonnées — Extraction EXIF | ★★★☆☆ |
| 09 | GOBUSTER — Énumération répertoires | ★★★★☆ |
| 10 | Google Dorks — Recherche avancée | ★★★★☆ |
| 11 | Sous-domaines — Énumération DNS | ★★★★☆ |
| 12 | CTF — Challenge final OSINT | ★★★★★ |

Les scores sont sauvegardés localement (`.osint_scores`).

**VPN** *(vpn.sh)*
- Statut en temps réel (interface `tun0` + IP publique)
- Ajout de profils `.ovpn` (colle, copie locale ou URL)
- Connexion OpenVPN en daemon (background)
- Déconnexion propre
- Consultation des logs OpenVPN
- Test IP publique avec géolocalisation
- Installation automatique d'OpenVPN si absent

**GitHub**
- `git pull` pour récupérer les mises à jour
- `git push` avec commit automatique horodaté
